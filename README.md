# Wizascript

All-in-one UnderScript plugin suite for Undercards.

## Structure

- `wizascript.user.js` — the userscript. This is the only file players need to install.
- `packages/core` — shared utilities used by every feature below.
- `packages/patch-maker` — custom Patch Notes overlay and editor.
- `packages/true-hub-bridge` — True Hub deck browser overlay for the Classic Hub page.
- `packages/deck-tracker` — in-game match-state tracking HUD.
- `bot/` — Discord bot scripts that produce the deck data used by True Hub Bridge.

## Status

Early scaffolding. Not yet functional.
